# Mini Flip Calendar 2026

Printable mini flip calendar for 2026 in English.

The repo is intentionally static:

- `index.html` is the ready-to-use layout
- `assets/characters/` contains the linked monthly illustrations
- `mini-flip-calendar-2026.pdf` is the printable export committed with the current version

Open locally:

```bash
open index.html
```
